Revenant

Escaneo con nmap, encuentra un sevidor samba, un RDP y varios servidores web ISS. Pare ser un windows server 2016

 

We discover that there is a shared Disk on the SMB server called nt4wrksv, and we will try to connect to enumerate it further.

smbclient \\\\10.10.x.x\\nt4wrksv

Tras intentar usar estos credenciales para acceder al RDP o a otras carpetas de samba parece que son un señuelo.

Así que se prosigue enumerando con gobuster hasta que se encuentra el directorio nt4wrksv en la web de ISS del puerto 49663. Y tras otra enumeración se encuentra el mismo archivo que en el samba. Por lo que podemos pensar que estamos en la misma carpeta.
